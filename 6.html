오늘 배운것

  1. style태그는 css언어의 경계를 표현하는 태그로써 css언어가 사용되는 범위의 처음과 끝에 써준다

  2. a{
    color:black;
    }

    a:선택자(selector)이 웹페이지에 있는 모든 a태그를 선택한다는 뜻
    color:black;:선언(declaration) 선택자가 지정하는 태그들에 대해서 어떠한 효과를 줄것인가에 해당되는 것
    color:속성(property)
    red:property의 값(value)
  3. text-decoration: none;:글자 장식(밑줄)없애기
  4. text-decoration: underline:밑줄 긋기

KEY POINT

  1. a{
    color:black;
    }
    :범위를 지정한 모든 태그들의 상태를 바꾼다

  2. a href="3.html" style="color:red; text-decoration:underline">
    :css태그를 쓴 <> 안에있는 태그들의 상태만 바꾼다

    *단 외박을 제외한 횟수임
  7. 2번 사항은 한달에 최대 1번 면제 가능하다.
