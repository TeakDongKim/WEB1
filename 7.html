  1. h1{pont size:45px} 글씨 크기 조정

  3. a href="index.html class="saw">: 이 태그를 saw라는 선택자의 class로 지정
  4. .saw{color:gray} class 네임 앞에는 . 이 무조건 잇어야한다
  5. 클래스라는 속성의 값은 여러 개의 값이 들어올 수 있고 띄어쓰기로 구분한다.
  6. 하나의 태그에는 여러개의 속성이 들어올 수 있고, 여러개의 선택자를 통해서 하나의 태그를 공동으로 제어할 수 있다.
  7. 가까이에 있는 명령이 더 큰 영향력을 갖는다 ex/ class 선택자가 중복된다면 밑에있는 명령만 실행됨
  8. a href="7.html" id="active"> 이 태그를 active 라는 선택자의 id로 지정
  9. #active{color:gray} id 선택자 앞에는 #이 무조건 있어야한다.
  10. 영향력 순서 = id선택자 > class선택자 > tag선택자
  11. 이미 한번 쓰여진 id의 선택자는 다시 사용하면 안된다 ex/ 한사람의 id카드는 1개 이기때문이다. (학번) 중복되어서는 안됨

KEY POINT

  1. class 태그 앞에는 무조건 . 이 있어야 한다.
  2. id 태그 앞에는 무조건 # 이 있어야 한다.
  3. 영향력 순서 id선택자 > class선택자 > tag선택자
  4. 태그는 서로 묶어서 쓸 수 있다.
  5. id 태그는 중복이 불가능하다.
  6. border은 테두리 이다.
  7. grid 사용법: div로 그룹지정후 id를 지정하고 grid를 사용한다.
