WEB1-10 ~ WEB1-14

오늘 배운것
  1. 속성: ex/ img src="coding.jpg"width="100%" img는 이미지 src는 source ""안에 있는것은 사진 이름 width는 크기를 나타낸다.
  2. li: 목차(list) 자식태그
  3. ul: 목차 범위 지정 (unordered list) list의 부모태그
  4. or: 목차 범위&순서 지정 (ordered list) list의 부모태그
  5. title: 웹 페이지 제목 지정(책표지)
  6. body: 본문은 body로 묶기로 약속(고위직 태그)
  7. head:본문을 설명하는 태그들은 head로 묶기로 약속(고위직 태그)
  8. meta charset="utf-8: 웹브라우저에게 utf-8로 문서를 읽으라고 지시(character:문자 & set:규칙 )
  9. html: 모든 태그를 감싸는 제일 고위직의 태그
  10. !doctype html: 이 문서는 html이라는 관용적 태그
  11. a hreh="https://www.w3.org/" tatget="_blank" title="html speicification": hypertext는 링크라는 의미이다 이것의 첫글짜를 딴 h와 이 웹브라우저를 참조하라는 뜻을 가진 reference 앞의 3글자를 따서 ref 클릭했을 때 새 탭이 열리도록 하고싶은때는 target="_blank"를 사용 링크를 클릭하기 전에 무엇일지 알려주고싶다면 title="html15 speicification" "" 사이에는 설명하고싶은 문장 넣기
    12. 실습: 네이버 바로가기

    key point

    1. 자식태그는 항상 어미태그가 필요하고 어미태그는 항상 자식태그가 필요하다
    2. !doctype html 과 html 태그는 사람들과의 약속이기 때문에 반드시 사용한다
    3. title 태그는 검색엔진에서도 유리한 결과를 얻도록 해주기 때문에 반드시 사용한다
    4. meta charset="utf-8 태그는 한글이 쓰여지지 않을때 사용할수있다
    5. 이미지를 불러올때는 web 파일로 가져와야한다

    오늘의 일기


    오랜만에 친구들과 나들이를 가니 기분이 들떳던 날이었다. 박물관들은 별 감흥이없었다. 하지만 현대 제철 공정과정을 보고 설명을 들으니 과연 내가 이런 엄청난 기업과 상대가 가능할까 라는 생각이 들었다. 하지만 난 내가 반드시 이길수 있을것이라고 믿는다. 이런 생각을 할때마다 매일 드는 생각이 내가 너무 허황된 꿈을 꾸는것이 아닌가 내가 이 길을 가는게 맞는것인가. 세상을 너무 쉽게 보는것이 아닌가 하는 생각이 들지만 그래도 난 자신감을 갖고 도전 할것이다. 그래 현재의 나는 보잘것없는 능력의 사람이지만 난 꿈과 의지를 가진 멋진 사람이다. 그렇기 때문에 언젠가는 반드시 성공할 것이라고 믿고 노력할것이다. 코딩 공부를 어떻게 해야 좋을지도 아직 잘 모르겠다. 일단 유튜브로 하는게 나을것같다. 오늘은 2시반 30분 남짓 코딩공부를 했는데 졸린건지 지루한건지 잘 모르겠다. 이게 내 적성에 맞는지는 더 해봐야 알 것같다. 그래도 내가 2시간 30분동안 무언가 공부한것은 굉장히 드문 일이고 어제 내가 하기로한 분량은 마무리했기 때문에 그래도 괜찮은 상황인것같다. 오늘은 고기 사오기와 헬스장 가기라는 어제의 목표를 실패했다. 벌로 내일은 유산소를 10분이라도 뛰어야겠다. 이젠 미루지말고 피하지말고 매일의 목표를 달성하는 내가 기대된다. 그때가오면 나도 무엇이든 성공할수있을것같다. 나를 실망시키지 않기 위해, 내일은 꼭 목표를 달성해보자 난 할수있어 화이팅^^
